🖥️ Setting Up Virtualization (VirtualBox / VMware)
📌 Install a virtualization tool to run Linux in a safe, isolated environment.

🔹 Steps to Install VirtualBox
- Go to the VirtualBox official website.
- Navigate to Downloads and choose your operating system (Windows, macOS, or Linux).
- Download the installer package.
- Run the installer and follow the setup wizard (default options are fine for beginners).
- Once installed, launch VirtualBox from your applications menu.

🔹 Next Steps to download Ubuntu
- Download an ISO image of Ubuntu 22.04 LTS from Ubuntu’s official site.
- Create a new VM in VirtualBox or VMware.
- Attach the Ubuntu ISO and start the installation.
- Once installed, you’ll have a fully functional Linux VM to practice commands and scripting.

🔹Step to Create a New Virtual Machine
- Open VirtualBox.
- Click New.
- Name it something like Ubuntu 24.04.4.
- Set Type: Linux and Version: Ubuntu (64-bit).
- Click Next.
- Instead of the normal wizard, check the option “Skip Unattended Installation” is unchecked — this enables unattended mode.

How to Open Your Ubuntu VM
- Launch VirtualBox on your computer.
- In the list of virtual machines, select the one you created (e.g., Ubuntu 24.04.4).
- Click Start (green arrow button).
- The VM will boot up, showing the Ubuntu login screen.
- Enter the username and password you set during installation.
- You’ll now be inside your Ubuntu desktop environment.
